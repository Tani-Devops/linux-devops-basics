# Linux Devops Basics 

## Overview 
This repository contains hands-on Linux fundamentals for aspiring DevOps engineers. 
It covers: 
- Linux filesystem structure
- File permissions and ownership
- Hard and soft links 
- Bash scripting for automation

## Day 1 - Filesystem & Permissions 
- Explored '/', 'etc', '/var', '/home', 'usr'
- Learned numeric and symbolic file permissions 
- Practiced creating secure directories
- Experimented with hard and soft links 

## Purpose 
These exercises simulate real-world DevOps tasks and helps develop production-ready Linux skills. 

## Usage 
1. Explore the 'notes/' folder for explanations. 
2. Explore the 'scripts/' folder for Bash scripts.
